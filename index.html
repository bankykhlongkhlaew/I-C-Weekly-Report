📖
วิธีติดตั้งและใช้งาน

ขั้นที่ 1 — ตั้งค่า Google Apps Script

  1. เปิด Google Sheets ใหม่ → ตั้งชื่อ "IC Weekly Report 2568"
  2. เมนู Extensions → Apps Script
  3. วาง Code จากไฟล์ Code.gs ลงใน Editor
  4. กด Deploy → New Deployment → Web App
  5. ตั้ง "Who has access" = Anyone → Deploy → คัดลอก URL

ขั้นที่ 2 — ตั้งค่า config.js

// config.js — แก้ค่านี้หลัง Deploy
const SCRIPT_URL = "https://script.google.com/macros/s/YOUR_ID/exec";
const SHEET_NAME = "IC_Weekly_2568";

ขั้นที่ 3 — อัปโหลดขึ้น GitHub Pages

  1. สร้าง Repo ใหม่ใน GitHub → อัปโหลด index.html + config.js
  2. Settings → Pages → Branch: main → Save
  3. URL จะเป็น https://YOUR-NAME.github.io/ic-report/
  4. แชร์ URL ให้ทีม — เปิดได้ทุก Device ทุก Browser

วิธีใช้รายสัปดาห์

📝 ทุกวันจันทร์
1. เปิด URL → กรอกข้อมูลสัปดาห์ที่ผ่านมา
2. กด Preview ตรวจสอบ
3. Submit → ข้อมูลบันทึก Google Sheets อัตโนมัติ
📊 สิ้นเดือน
เปิด Google Sheets → Tab "Monthly_Summary"
จะ Auto-aggregate ข้อมูลทุกสัปดาห์ในเดือนนั้นให้อัตโนมัติ
⚠️ หมายเหตุ PowerEx: ค่า Protection System Readiness ต้องกรอกเองโดยดูจาก PowerEx บน EGAT WiFi — ระบบไม่สามารถดึงข้อมูลจาก 10.212.28.3 โดยอัตโนมัติได้เพราะเป็น Private IP Network
